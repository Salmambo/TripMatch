## 5. 실행 방법

- 백엔드:
  ```bash
  1. cd server
  2. npm install
  3. npm run dev
  4. 브라우저에서 localhost:5000/api-docs 접속
  5. Swagger 문서에서 API 선택하여 Try it out
  6. Request 양식 채워서 Execute
  ```
